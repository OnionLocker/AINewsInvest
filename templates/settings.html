配置指南
  1. 在 Telegram 中搜索 @BotFather，发送 /newbot 创建新机器人
  2. 复制 BotFather 回复的 Bot Token 填入上方
  3. 在 Telegram 中找到你刚创建的机器人，发送任意一条消息
  4. 浏览器访问 https://api.telegram.org/bot你的Token/getUpdates
  5. 在返回的 JSON 中找到 chat.id 字段，填入上方 Chat ID
安全说明

你的 Bot Token 使用 Fernet (AES-128-CBC + HMAC) 对称加密后存入数据库， 明文 Token 不会被直接保存。每次发送消息时实时解密使用。
